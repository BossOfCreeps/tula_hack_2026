# Задание

TEAM BUILDER  
(сервис для руководителя по формированию профиля сотрудника)

## Проблематика:

Руководитель не всегда понимает какой сотрудник ему нужен в команду для эффективной и слаженной работы, а также определения условий и инструментов мотивации по работе с действующим составом

## Задача:

Разработать сервис-помощник (конструктор) для руководителя при помощи которого руководитель сможет формировать профиль для подбора и расстановки сотрудников в своей команде, а также получать готовые рекомендации по работе с ними и созданию необходимых условий

## Механика:

Руководитель имеет возможность на основании критериев:
- роль в команде
- типология личности (DISC)
- мотивационный профиль
- возраст/поколение

Получить рекомендованный профиль для подбора сотрудника либо по работе с действующим составом при известных вводных (текущий состав и свой профиль как руководителя)

Выбор инструментов и способ реализации оставляем за рабочей группой.
